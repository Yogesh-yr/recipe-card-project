# Classic Chocolate Fudge Cake — Interactive Recipe Card

An interactive recipe card built with HTML, CSS, and JavaScript.

## Files

- `index.html` — page structure (title, photo, ingredients, method, prep time, serving size)
- `style.css` — box-model layout, hover effects, transitions, animated progress bar, responsive media queries, print styles
- `script.js` — toggle visibility, "Start Cooking" step sequencing, cake-layer progress updates, countdown timer, print
- `assets/cake-photo.jpg` — recipe photograph
- `preview.html` — the same page with CSS/JS/photo inlined, for quick one-click viewing

To run: open `index.html` in any browser. No build step or server required.

## Features

- Recipe card layout with title, photograph, ingredients list, numbered method, and prep time / serving size, structured with the CSS box model
- Hover effects and smooth transitions on ingredient and step rows
- An animated cake-layer progress bar that fills as cooking steps are completed
- "Show Ingredients" / "Show Steps" toggles
- "Start Cooking" mode that highlights each step in sequence, with a "Next step" button and a "Reset" option
- Fully responsive layout across mobile, tablet, and desktop
- A 30-minute countdown timer that starts automatically when cooking begins
- A print-friendly stylesheet for a clean printed copy of the recipe
